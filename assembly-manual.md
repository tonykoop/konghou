# Konghou Mule Assembly Manual

## Before Cutting

1. Review `design.md`, `drawing-brief.md`, and `validation.csv`.
2. Replace any stock sizes in `cut-list.csv` that do not match the available material.
3. Confirm tuning pin, hitch pin, and string supplier data.
4. Record real string unit weights in `string-schedule.csv`.

## Single-Row Mule Sequence

1. Make pin and anchor test coupons from offcuts.
2. Drill and test the tuning pin pilot size before drilling the neck rail.
3. Fit the lower frame rail, upper neck rail, and compression column as a dry assembly.
4. Clamp the soundboard test panel and bridge rail in the intended load path.
5. Add strings one at a time while logging deflection after each pair of strings.
6. Stop immediately if any joint opens, a pin slips, the bridge lifts visibly, or the soundboard cracks.
7. Run the 50 percent, 100 percent, and 125 percent load gates in `validation.csv`.
