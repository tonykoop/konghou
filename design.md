# Konghou Starter Design Notes

Current status: `bare-bones readiness packet`.

Fabrication authority: `not build-ready`. This file records a public starter
packet and evidence gaps. It is not a measured construction plan, a CAD release,
or a safe-to-string frame design.

## Intent

Create a first honest repo packet for a konghou-inspired harp/zither-family
instrument. The packet is meant to make decisions and unknowns visible before
any builder commits to string gauges, frame proportions, bridge placement,
soundboard thickness, joinery, or high-tension hardware.

## Provenance and Inspiration

The working design family is inspired by Chinese konghou history and by the
broader arched-harp / frame-harp / board-zither problem space:

- Historical inspiration: konghou silhouettes, vertical harp posture, visible
  string courses, and ornamental frame language.
- Comparative instrument families: harps for high string tension and frame
  loading, zithers for soundboard/bridge transfer, and experimental hybrid
  layouts for repeatable maker documentation.
- Build authority: future CAD/DXF files, measured templates, string tension
  calculations, and validation-loop evidence only.

Historical images, museum descriptions, AI-generated concepts, or aesthetic
sketches must not be treated as dimensional evidence.

## Public-Readiness Gates

The public packet should keep these labels until evidence changes:

- Current status: `bare-bones readiness packet`.
- Measurement status: `measurement-required` for scale lengths, string spacing,
  soundboard thickness, bridge compensation, frame section sizes, and anchoring
  hardware.
- CAD/DXF status: `future authority unless measured geometry exists`.
- Sourcing status: `unverified until checked at purchase time`.
- Safety status: `load-test-required` before full tensioning.

Promotion to a shop packet requires a reviewed drawing set and a validation log
that proves the frame and soundboard can tolerate the selected strings.

## Design Assumptions

Starter assumptions are placeholders, not build instructions:

- String courses: single-course melody strings first; multi-course or sympathetic
  strings are future scope.
- Pitch plan: moderate diatonic or pentatonic range first, chosen after actual
  scale length and string material are known.
- Soundboard: thin resonant board or laminated plate, thickness
  `measurement-required`.
- Frame/neck: rigid frame with pillar or equivalent compression member,
  cross-section `measurement-required`.
- Bridge/nut: pinned or bearing bridge plus nut/guide surface, placement
  `measurement-required`.
- String material: nylon, fluorocarbon, gut, brass, bronze, or steel remain
  candidate options until tension calculations and supplier ratings are known.
- Tension assumption: total frame load is unknown. It must be calculated from
  string unit weight, speaking length, target pitch, and selected gauge before
  hardware purchase or fabrication.

## Historical Inspiration vs Build Authority

Allowed uses of historical inspiration:

- Silhouette studies.
- Ornament, finish, and cultural context notes.
- Comparing playing posture and string access.
- Naming constraints and public documentation provenance.

Not allowed as build authority:

- Inferring scale lengths from images.
- Copying frame proportions from paintings or museum photos without measured
  dimensions.
- Treating visible string count as a safe tension plan.
- Using concept art as bridge, pin, or toolpath geometry.

## Safety Notes

String instruments store energy. A frame, tuning pin, string anchor, or string
can fail suddenly during tensioning. Before a full prototype:

- Use eye protection and keep hands/face out of the string plane while tensioning.
- Stage load gradually and record deflection after each load step.
- Add temporary shielding for first tension tests.
- Use only hardware with known pull-out and shear capacity once actual tension
  is calculated.
- Retire strings after visible kinks, corrosion, fraying, or overstretch.
- Stop if frame twist, soundboard bellying, bridge lift, pin creep, or joint
  opening appears.

## Open Measurements

These values are intentionally not filled in yet:

- Number of strings and final tuning map.
- Per-string speaking length.
- Per-string material, gauge, unit weight, and target tension.
- Total frame load and load path.
- Frame member cross-sections and joinery.
- Soundboard material, grain direction, thickness, bracing, and bridge footprint.
- Bridge/nut radius, bearing angle, string spacing, and compensation.
- Tuning pin, hitch pin, and anchor ratings.
- Player ergonomics: lap/stand height, reach, and hand clearance.

## Next Practical Step

The next useful build artifact is a tension and frame-load study tied to a small
set of candidate string materials, followed by a low-cost test frame or
soundboard coupon. CAD/DXF should follow measured decisions rather than lead
them.
