# Modern Konghou Mule-First Promotion

# Project Intent

Promote a modern konghou-inspired paired-course family while keeping the first physical work conservative: a single-row structural mule proves load path, tuning access, and measured string data before any final double-row build.

# Deliverables

- Single-row mule packet with DXF review layout.
- BOM, cut list, sourcing, and string schedule.
- Proof-load validation gates.
- Mule-to-family frame and string scaling ledger.
- Concept-image rules that keep generated images out of fabrication authority.

# Readiness

L2 prototype scaffold. The packet is ready for review and mule planning, not final paired-course fabrication.

# Promotion Blocker

`KGH-PAIR-01` remains blocked until `KGH-MULE-01` passes every proof-load and measured-update row in `validation.csv`.

# Validation

The initial validation target is structural: 50 percent, 100 percent, and 125 percent static load steps with measured neck lean, bridge lift, and soundboard crown change.

# String Evidence

The string schedule includes target notes and starting speaking-length envelopes only. Calculated tension cells remain empty until vendor or measured unit weights are recorded.

# Frame/String Scaling

`frame-string-scaling.csv` blocks the paired-course path until mule tension,
scale fan, row spacing, frame section, bridge/soundboard transfer, and visual
authority rows have measured or reviewed evidence.

# DXF Authority

The included DXF is a mule review scaffold only. It must be replaced before paired-course fabrication.

# Visual Authority

Concept images may explain the program, but they cannot define dimensions, cut files, pin locations, or bridge geometry.

# Safety Gates

Stop the mule test on visible joint opening, pin slip, bridge lift, soundboard cracks, or brace release.

# Serviceability

Tuning pins and anchors must remain reachable with actual tools before paired-course spacing is promoted.

# Open Decisions

Range, scale fan, final string material, soundboard species, brace pattern, and row spacing remain open until measured mule data exists.

# Would Route Here Again

Yes. Future konghou work should route to this packet because it keeps the modern paired-course goal and the mule gate in one canonical repo location.

# Next Actions

Replace scaffold assumptions with real supplier string unit weights, measured mule geometry, and revised CAD/DXF after proof-load testing.
